# Excel-Project
Investment Analysis Dashboard

This project focuses on analyzing investment behavior using survey data. The dataset contains information about individuals' demographics, investment preferences, savings objectives, and investment duration. The goal of the project is to identify patterns in how different groups of people invest and save money.

The dashboard provides insights into:

Demographic Analysis – Understanding investment trends based on gender and age groups.

Investment Preferences – Identifying the most preferred investment options among individuals.

Savings Objectives – Analyzing why people invest (wealth creation, safety, retirement, etc.).

Investment Duration – Understanding how long people prefer to keep their investments.

Behavioral Patterns – Observing how age groups influence saving and investment decisions.

Tools & Technologies Used

Excel – Data cleaning and dashboard creation

Data Visualization – Charts and graphs to present insights clearly

Project Objective

The objective of this project is to transform raw survey data into meaningful insights through data analysis and visualization, helping understand investor behavior and financial decision-making patterns.
